读书笔记-《我的互联网方法论》

1.没有人能够打败趋势

反过来说，就是要看清局势，顺势而为，选择大于努力。

2.用户是谁？会遇到什么问题？对手有没有解决？我们怎么做得更好？

前两个问题就是我们在《银行4.0》中提到的第一性原理，直面问题并解决问题。
后两个问题则是我们在《如何提高核心竞争力》中讲到的，向领域内优秀者学习，找到自己的竞争力组成要素并为之努力，从而拥有核心竞争力。

3.用户不知道你 -> 价值链边缘化

如果在业务开展中，你并不能直接触达到用户，那么你就被边缘化了，减少了获利机会。

4.互联网盈利方式：卖东西、广告收入、增值服务


5.从传统客户思维到互联网用户思维

传统商业模式是，我把东西卖给客户，并向他提供服务和产品。
而互联网模式是，我会向用户长期提供服务，并保持联系，只有在互联网上积攒了很多用户后，才有能力将其中一部分转换为客户。

6.在一个点上追求极致，做到世界第一的小事，就是大事


7.关于创新

商业模式上创新：贵 -> 便宜， 收费 -> 免费。
体验上创新：复杂 -> 简单。

8.超出预期的才是用户体验，体验的基础是用户需求


9.像小白一样思考

不要用“身边统计学”，读了大学就认为人人都有本科学历，开始工作就认为自己的专业知识是通识。
实际上隔行如隔山，大多数人都是小白，不明白也不需要明白那么多技术细节。

10.做让用户离不开的产品


11.学会小步快跑
